# yt-dlp Batch Downloader Logs

This folder contains logs generated by the yt-dlp batch downloader script during encoding sessions.

Each log file is named with the format:
`encoding_YYYY-MM-DD_HH-MM_MODE_PROFILE.txt`

Where:
- **YYYY-MM-DD**: The date of the encoding session
- **HH-MM**: The time the session started
- **MODE**: GPU, CPU, or NOENC (no encoding)
- **PROFILE**: The encoding profile used (STANDARD, HQ, LIVEACTION, ANIMATION, MIXEDVIDEO, MEDIUM, SLOW, NONE)

## Log Contents

Each log file contains:
- Session metadata (date, time, mode, profile, browser, download directory)
- Per-video processing information:
  - URL being processed
  - Detected codec type (PREMIUM, AV1, VP9, H264)
  - YouTube Premium subscription status (if detected)
  - Action taken (download only or download + re-encode)
  - Success/failure status
- Summary statistics at the end of the session
